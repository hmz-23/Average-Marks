# Average marks of a Student

#### In this program we have to take input & store marks of 3 subjects to display the average and the Remark that the student got.

#### * Average Marks = Total Marks / 3

| Average Marks  | Remark |
| ------------- | ------------- |
| 85 - 100  | EXCELLENT  |
| 75 - 84  | DISTINCTION |
| 60 - 74  | FIRST CLASS |
| 40 - 59  | PASS |
| Less Than 40  | FAIL  |
